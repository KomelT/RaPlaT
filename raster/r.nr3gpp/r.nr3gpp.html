DESCRIPTION

r.nr3gpp computes a path-loss raster from a single transmitter using simplified large-scale formulas based on 3GPP TR 38.901 for:
  • uma (Urban Macro)
  • umi (Urban Micro Street Canyon)
The output raster stores path loss in dB.

NOTES

  • Input frequency is in MHz (internally converted to GHz).
  • Path loss is calculated for distances from 10 m up to radius.
  • The model is large-scale only; it does not include clutter/building diffraction by itself.

EXAMPLE

r.nr3gpp input_dem=dem_map output=nr_uma \
  scenario=uma coordinate=460697,99918 ant_height=25 rx_ant_height=1.5 \
  frequency=3500 radius=10
